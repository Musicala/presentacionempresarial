# Alianza artística comunitaria · Musicala

App web tipo brochure para reuniones de alianza.

## Uso
Abre `index.html` en el navegador. No necesita instalación ni internet.

## Archivos
- `index.html`: estructura y contenido.
- `styles.css`: diseño visual Musicala.
- `script.js`: navegación, progreso, menú móvil y animaciones.
- `logo.png`: logo conectado en la interfaz.

## Ajustes rápidos
- Cambiar textos: edita las secciones dentro de `index.html`.
- Cambiar colores: edita las variables `:root` en `styles.css`.
